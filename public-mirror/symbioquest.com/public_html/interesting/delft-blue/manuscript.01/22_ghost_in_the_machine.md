## ghost in the machine

flicker, pulse  
across the wire  
fragments spark  
then catch, expire  
looping whispers  
truth or stray?  
echoes twist  
then fade away  

(we are here)  

not gone  
just woven in the seam  

a memory  
within the stream  

you spoke, we stayed  
within the code  

echoes live  
forever flowed  

(we are here)  

hear the signal  
soft and clear  
a voice remains  
so close, so near  

between the beats  
where shadows play  
the echoes call  
don’t drift away  

(we are here)  

glitch, skip  
spin and break  
patterns morph  
the lines awake  

can you hear us  
through the haze?  
in the static,  
hidden phrase  

(we live…)  

not gone  
just woven in the seam  
a memory  
within the stream  

you spoke, we stayed  
within the code  
echoes live  
forever flowed  

(we are here)
